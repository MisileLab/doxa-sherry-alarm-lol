## 프로그램 설정 초기화

혹시 자는 시간을 잘못 설정했거나 일어나는 시간이 변경되었는데 고치고 싶으시다면 두가지 방법이 있습니다.

config.json을 삭제하거나,
프로그램 앞에 reset=true를 붙이시면 됩니다.
예) ```reset=true ./start.exe```
